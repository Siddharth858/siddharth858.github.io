Description of files

  1. testing_files : This folder contains Testing ham and spam mails from enron dataset.
  2. training_files : This folder contains Training ham and spam mails from enron dataset.
  3. ham_classifier.py : Finding posterior probability of a word given that it is ham mail.
  4. posterior_probability_word.py : It contains two functions to calculate posterior probability.
  5. spam_classifier.py : Finding posterior probability of a word given that it is spam mail.
  6. spam_classifier_data.txt : Text file to store posteriors probabilities (p(word|spam)).
  7. testing_complete.py : This file tests all the spam messages in the testing folder.
  8. test_singlemail.py : Put the mail in text file named input_new_mail_to_test.txt, Run this file to test a single mail if it is spam or not.
